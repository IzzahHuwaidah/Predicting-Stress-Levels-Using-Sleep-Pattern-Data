# Predicting-Stress-Levels-Using-Sleep-Pattern-Data
Proyek Data Science menggunakan Machine Learning untuk memprediksi tingkat stres melalui analisis pola tidur. Fokus pada metrik durasi &amp; kualitas untuk deteksi dini kesehatan mental. Dikembangkan dengan Python (Pandas, Scikit-Learn) untuk EDA dan klasifikasi guna mendukung langkah preventif. Mencakup prapemrosesan &amp; evaluasi model..
